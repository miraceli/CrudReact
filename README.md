## 🛠️ Tecnologias

As seguintes tencologias foram usadas no projeto:

> - NodeJS


Antes de começar, você precisa ter instalado em sua máquina as seguintes ferramentas:
> - [Git](https://git-scm.com)
> - [NodeJS](https://nodejs.org/en/)
 
📌 Além disto é bom ter um editor para trabalhar com o código como [VSCode](https://code.visualstudio.com/) e uma plataforma para testar requisições como [Insomnia](https://insomnia.rest/download) . O ambiente de trabalho utilizado foi o Windows e os comandos executados pelo [Windows Terminal](www.microsoft.com/pt-br/p/windows-terminal/9n0dx20hk701).

&nbsp;

## 🧩 Preparando o ambiente

```bash
# Verifique a versão do Node
$ node -v

# Verifique a versão do NPM
$ npm -v

# Verifique a versão do Yarn
$ yarn --version

# Instale globalmente o Expo 
$ npm install --global expo-cli

# Se necessário, atualize todas as dependências do NPM
$ npm update -g
$ npm audit fix --force
```

&nbsp;

## 🎲 Rodando o backend

```bash
# Clone este repositório
$ git clone <https://github.com/miraceli/CrudReact>

# Acesse a pasta do projeto backend no terminal/cmd
$ cd backend 

# Execute a aplicação 
$ python index.py
```
